---------------------------------------------------------

Constantes:
  MAX_CAPACIDAD: máximo valor que puede tomar la variable capacidad (+1)
  MAX_OPT: cantidad de opciones del menu de set up (capacidad, limite de aviso, escape)

Variables: 
  bool bloqueado:   se inicia en false
  bool vaciar:      se inicia en false
  int capacidad:    se inicia en MAX_CAPACIDAD - 1
  int lim_aviso:    se inicia en capacidad - 1
  int cant_autos:
  int param_index:

Eventos:
  activar
  desactivarUp
  desactivarDown
  ingreso
  egreso
  vaciarUp
  vaciarDown
  config_enter
  next

Estados:

El sistema general tiene 3 estados:
  ST_Desactivado
  ST_Normal
  ST_SetUp

El estado inicial es Desactivado. El sistema se activa con un pulsador que solo tendrá efecto si el switch de bloqueo no esta activo.

En cualquier momento de la ejecución el sistema se desactiva mediante un switch, bloqueando la posibilidad de activación hasta que el switch vuelva al estado bajo.

Estando en ST_Normal se puede pasar a ST_SetUp mediante el pulsador de configuración, en este estado se podrán modificar las variables de [capacidad] y [lim_aviso]

De ST_SetUp se sale por el [Escape] del main y se vuelve a ST_Normal entrando por un punto distinto según la configuración establecida.

task_normal: (funcionamiento del modo normal)
  La entrada por defecto setea [cant_autos = 0] y el estado inicial es ST_Norm_Vacio. Al llegar un ingreso se pasa al estado ST_Norm_NoVacio.
  
  En ST_Norm_NoVacio:
    Se registran ingresos y egresos mientras [1 < cant_autos < lim_aviso].
    Si se produce el egreso del último auto se vuelve a ST_Norm_Vacio.
    Si se produce el ingreso que resulta en [cant_autos == lim_aviso] se pasa a ST_Norm_Aviso.

  En ST_Norm_Aviso:
    Se registran ingresos y egresos mientras [lim_autos <= cant_autos < capacidad].
    Si se produce un egreso cuando [cant_autos == lim_aviso] se vuelve a ST_Norm_NoVacio.
    Si se produce un ingreso que resulta [cant_autos == capacidad] se pasa a ST_Norm_lleno

  En ST_Norm_Lleno:
    ...

  En cualquier estado se puede activar un switch que fuerza al sistema a volver a ST_Norm_vacio seteando [cant_autos = 0] y manteniendolo así hasta que se desactive el switch.

task_set_up: (funcionamiento del modo set up)
  El menu se maneja con dos pulsadores configuracion/enter y next.

  El estado inicial es ST_SetUp_Main con [param_index = 0] mediante el pulsador next se cicla el indice del menu. El indice en 0 accede a la configuracion de capacidad, en 1 accede a la configuración del límite para el aviso y en 2 es el escape para salir del modo set up. Con el pulsador configuracion/enter se accede a la configuración seleccionada.

  En ST_SetUp_Capacidad se cicla la variable [capacidad] entre 0 y MAX_CAPACIDAD-1. Se confirma con configuracion/enter y se vuelve a ST_SetUp_Main

  En ST_SetUp_Aviso se cicla la variable [lim_aviso] entre 0 y [capacidad]-1. Se confirma con configuracion/enter y se vuelve a ST_SetUp_Main
